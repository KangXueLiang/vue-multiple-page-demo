# vue3.*  (多页面跳转)

 @[vue2.5.2|webpack3.6.0]

>- 空白的脚手架,配置文件增加了中文注释,assets中配置了部分公共的内容,拉下来直接开发
>- 在static的tpl文件夹中有空的多页面文件夹模板,开发时直接复制修改一下名字即可

## 命令
>国际惯例

- **npm install (or cnpm 其他)** 装依赖包
- **npm run dev** 起服务
- **npm run build** 打上线包
- 跑起来服务后,需要在url输入 /*/*.html (*为对应文件夹名字)
- 添加了新文件夹后,需要重新 npm run dev一下
